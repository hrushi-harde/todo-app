# Full Stack To-Do App

## 🚀 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: FastAPI
- Database: PostgreSQL

## ✨ Features
- Add tasks
- Delete (complete) tasks
- Real-time UI updates
- Persistent storage using PostgreSQL
- Toast notifications

## ⚙️ Setup

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
